# SOE Report Generation Skill

## Purpose

Guide generation of management reports, compliance checklists, and project
summaries for Sunrise Ocean Engineering Limited using data stored in the
SOE AI Commercial Agent database.

## Scope

- Company file map (all documents by type and project)
- Missing document report
- Risk register (filtered by project, risk level, status)
- Accounting draft report (by status and draft type)
- Tax compliance report (flagged documents)
- QS / commercial summary
- Monthly closing checklist
- Project summary report

## Out of Scope

- Official financial statements (requires CPA preparation)
- Audited accounts
- Tax returns
- QS certified reports (requires qualified QS)
- Legal opinions

## Report Types and Data Sources

### Company File Map
```
Source: Document table, grouped by document_type and project_id
Output: {project → {document_type → [documents]}}
Key metrics: Total documents, missing required types, unreviewed count
```

### Missing Document Report
```
Source: Risk table WHERE risk_category = 'MISSING_SUPPORT' AND status = 'OPEN'
Output: [{document_id, document_type, missing_item, project, risk_level, recommended_action}]
Priority: CRITICAL and HIGH risks first
```

### Risk Register
```
Source: Risk table, optionally filtered by project_id, risk_level, status
Output: Sorted by risk_level DESC, then created_at DESC
Columns: risk_title, risk_level, document, project, requires_professional_review, status
```

### Accounting Draft Report
```
Source: AccountingDraft table
Output: {
  by_status: {DRAFT: count, PENDING_APPROVAL: count, APPROVED_FOR_EXPORT: count, EXPORTED: count},
  by_type: {BILL: count, SALES_INVOICE: count, ...},
  total_amount_by_currency: {USD: amount, HKD: amount},
  pending_cpa_review: [drafts where requires_cpa_review=True],
}
```

### Tax Compliance Report
```
Source: Review table WHERE agent_name = 'tax_agent'
Join: Risk table WHERE risk_category IN ('WHT_RISK', 'DGT_REQUIRED', 'PE_RISK')
Output: [{document, wht_applicable, dgt_required, cor_required, pe_risk, tax_risk_level, action_required}]
```

### QS Summary
```
Source: Review table WHERE agent_name = 'qs_agent'
Join: Document table WHERE document_type IN ('PAYMENT_CERTIFICATE', 'BQ', 'VARIATION_ORDER')
Output: {
  contract_sum_usd: ...,
  approved_vo_total: ...,
  revised_contract_sum: ...,
  cumulative_gross_certified: ...,
  cumulative_retention_held: ...,
  pcs: [{pc_number, period, gross, retention, net, status}],
  open_vos: [...],
  qs_risk_items: [...],
}
```

### Monthly Closing Checklist
```
Generated fresh each time — checks current DB state
Output: [{item, status: OK|MISSING|PENDING, details}]
```

## Monthly Closing Checklist Items

| # | Check | Pass Condition |
|---|---|---|
| 1 | All supplier invoices uploaded | No SUPPLIER_INVOICE risk with MISSING_SUPPORT open |
| 2 | All payment proofs uploaded | No PAYMENT_PROOF documents with PENDING_APPROVAL status |
| 3 | All PC / sales invoices reviewed | No PAYMENT_CERTIFICATE with UPLOADED or EXTRACTED status |
| 4 | All project costs assigned to project | No SUPPLIER_INVOICE with project_id = NULL |
| 5 | High-risk payments approved | No approval with PENDING status for HIGH/CRITICAL risk docs |
| 6 | Tax-sensitive items flagged | Tax agent review run on all cross-border invoices |
| 7 | Missing document list reviewed | All MISSING_SUPPORT risks ACKNOWLEDGED or RESOLVED |
| 8 | All accounting drafts reviewed | No AccountingDraft in DRAFT status older than 7 days |
| 9 | CPA review items exported | No draft with requires_cpa_review=True in PENDING_APPROVAL |
| 10 | Tax advisor review items exported | No open tax risk items without tax advisor flag |

## Compliance Limitations

**All reports must include disclaimer:** "Generated by SOE AI Commercial Agent based on uploaded document data. This is an internal management tool report only. It does not constitute a certified financial statement, audited account, legal document, or professional certification."

**Must NOT:**
- Present reports as audited or certified
- Omit the generation timestamp (reports are point-in-time snapshots)
- Include LLM prompt text or completion text in report output
- Include personally identifiable employee information beyond what is necessary

**Professional reports still required from:**
- HK CPA: audited financial statements, tax returns, management accounts
- Qualified QS: certified final accounts, VO assessments, extension of time
- Solicitor: legal contract opinions, dispute resolution advice
- Indonesia tax advisor: PPh filings, DGT applications, PE assessments
