## 1. 项目定位

### 对外推广钩子（唯一入口，用于论坛/社群/LinkedIn）
> 帮外贸工厂和贸易公司做英文产品网站，让海外客户通过 Google 找到你并提交询盘。

### 官网主标题
> 外贸网站建设 + 产品内容整理 + Google 基础 SEO + 询盘系统

### 官网副标题
> 专为工厂、贸易公司、设备供应商和 B2B 产品企业搭建英文产品网站，提供产品分类规划、英文文案整理、询盘表单、WhatsApp 联系、阿里云部署和后期维护。

### 服务延展（首屏之后再展示）
> 除外贸网站外，也支持企业官网、产品目录系统、报价系统、AI 客服和业务软件定制。

### 目标客户（优先级排序）
1. **外贸工厂 / 贸易公司 / 设备供应商** — 需要英文网站接海外询盘（最高优先）
2. **工程 / 建筑 / 装修公司** — 需要展示案例和资质（背景匹配）
3. **贸易 / 批发商** — 需要产品目录和询盘系统
4. **顾问 / 专业服务公司** — 需要预约和内容系统

### 核心差异点
不做模板建站。竞争优势在于：
- **内容整理服务**：帮客户整理公司介绍、产品分类、英文文案、FAQ——这是免费建站工具和低价代理商做不到的
- **完整部署**：阿里云 ECS + Nginx + SSL + SEO 提交，不只给文件
- **询盘导向设计**：每个页面围绕"让访客变成询盘"而非"展示好看"

### SEO 承诺边界
**可以说**：帮你搭建符合 Google 收录习惯的英文产品网站，完成网站结构、产品页面、SEO 标题、站点地图、Google Search Console 提交和询盘表单设置，让网站具备被搜索引擎发现和获得询盘的基础。  
**不说**：保证 Google 首页排名 / 保证一个月来询盘 / 保证百度 Google 一定搜到。

## 6. Demo 站规格

> **重要**：Demo 1 和 Demo 2 是推广钩子的直接支撑，必须做得最精致。

### Demo 1：外贸英文产品网站（优先级：最高 ★）
- **子域名**：`export.demo.yoursite.com`  
- **页面**：英文首页 · 产品分类 · 产品详情页（含参数表）· Inquiry 表单 · WhatsApp 按钮 · 公司介绍 · 证书展示 · 下载产品目录 PDF · FAQ  
- **SEO 示例**：展示 Google 搜索标题 / meta description 示例  
- **内容示例**：包含完整英文文案（非占位文字），体现"内容整理服务"价值  
- **商业价值**：最高，直接对应对外推广钩子

### Demo 2：产品目录 + 询盘系统（优先级：最高 ★）
- **子域名**：`catalog.demo.yoursite.com`  
- **功能**：产品分类 · 产品详情 · 参数表 · 加入询价单 · 提交询盘 · 后台查看询盘 · 产品管理后台截图  
- **商业价值**：体现"免费建站工具做不到的东西"，配合 Demo 1 形成完整外贸方案

### Demo 3：工程 / 建筑公司官网（优先级：很高）
- **子域名**：`engineering.demo.yoursite.com`  
- **页面**：公司介绍 · 服务范围 · 工程案例 · 项目照片 · 资质证书 · 合作客户 · 联系我们  
- **特性**：中英文切换  
- **商业价值**：与背景最匹配，适合香港工程 / 装修 / 分包商客户

### Demo 4：报价系统（优先级：高）
- **子域名**：`quote.demo.yoursite.com`  
- **功能**：选择产品/服务 · 自动计算价格 · 填写客户信息 · 生成报价单 · 导出 PDF · 报价记录 · 后台管理  
- **商业价值**：从"建网站"升级到"定制软件"的关键 Demo

### Demo 5：AI 客服 / 知识库问答（优先级：中高）
- **子域名**：`ai.demo.yoursite.com`  
- **功能**：上传公司资料 · AI 回答产品问题 · 留联系方式 · 人工接管提示 · 多语言问答 · 后台管理知识库  
- **注意**：强调"基于企业资料回答"，不做通用聊天机器人  
- **商业价值**：吸引眼球，适合作为升级服务

### Demo 6：专业服务公司官网（优先级：中）
- **子域名**：`consulting.demo.yoursite.com`  
- **页面**：服务介绍 · 预约咨询 · FAQ · 客户评价 · 文章知识库 · 表单收集资料  
- **适合**：会计 / 秘书 / 顾问 / 移民 / HR 服务公司

## 11. 外贸需求评估表（获客工具）
