# 微信 / 支付宝付款自动化方案

当前上线方案先采用“二维码收款 + 人工确认开通”：

1. 用户选择套餐。
2. 用户扫描微信或支付宝收款码付款。
3. 用户备注注册邮箱或订单号。
4. 用户在账户中心提交付款时间、付款人信息或截图说明。
5. 管理员核对到账后标记已付款，系统自动把用户套餐改为对应方案。

后续自动化开通建议按这个顺序做：

1. 申请微信支付商户号和支付宝开放平台应用。
2. 为两个网站分别配置支付回调地址。
3. 下单时由系统生成唯一订单号和金额。
4. 用户付款成功后，微信/支付宝向服务器发送异步回调。
5. 服务器验签，确认订单号、金额、状态无误。
6. 将订单标记为 paid，并自动开通套餐额度。
7. 记录支付流水、回调日志和开通日志，方便排查和退款。

上线自动化前不要只靠前端跳转判断付款成功，必须以后端回调验签为准。
