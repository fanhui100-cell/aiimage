# AI 变现实施计划（执行版 2026-05-20）

## 项目概述

- **目标**：6 个月内建立可持续收入流，从 AI 自动化服务外包起步，逐步沉淀标准交付包，再扩展到知识库、私有化和 Agent 定制。
- **核心原则**：先卖服务 -> 沉淀模板 -> 标准交付包 -> SaaS / 私有化。
- **执行重点**：前 2 个月只主攻 `n8n + Dify 自动化工作流`，简历工具和电商文案工具作为引流资产持续运行。
- **收入目标**：
  - Month 1：保底 ¥2000-5000，超预期 ¥8000+
  - Month 2：¥6000-15000
  - Month 3：¥15000-30000
  - Month 6：¥40000+

---

## 产品角色定位

| 产品 / 方向 | 当前角色 | 投入上限 |
|------|------|------|
| 电商文案工具（aicopytools.top） | 引流 + 电商客户入口 + 能力背书 | 每周优化 1-2 次 |
| 简历工具（aijobpro.top） | 小红书素材 + C 端付费测试 + HR 场景入口 | 每周维护即可 |
| n8n + Dify 自动化工作流 | Month 1-2 主收入线 | 每天重点推进 |

| 私有化 AI 部署 | Month 4+ 高客单方向 | 后期 |
| 短视频内容工厂 | Month 4+ 或有 GPU 资源 / 客户需求后启动 | 暂缓 |

## 主攻垂直场景

Month 1-2 只打 3 个场景，不泛化卖“AI 技术”。

### 1. 电商卖家

- 商品标题、详情页、卖点批量生成
- 客服自动回复
- 订单 / 售后表格处理
- 评价分析和差评归因

### 2. 小公司运营 / 自媒体团队

- 选题生成
- 文案排期
- 评论整理
- 周报 / 日报自动生成
- 内容自动发布或半自动发布

### 3. HR / 教培 / 咨询机构

- 简历初筛和 JD 匹配
- 内部资料知识库问答
- 课程资料整理
- 客户咨询自动回复

## 近期执行节奏

### Week 0：准备期（正式推广前 5-7 天）

目标：工具跑通、Demo 能展示、销售素材准备好。Week 0 不计入 Month 1。

#### 技术任务

- [ ] 服务器安装 n8n（Docker Compose）
- [ ] 服务器安装 Dify（Docker Compose）
- [ ] 配置一个稳定可用的 LLM API（优先 DeepSeek，复杂任务再用 Claude）
- [ ] 跑通 n8n -> Dify -> 输出结果的基础链路
- [ ] 确认日志、API Key、数据库和备份方式

#### 三个必须跑通的 Demo

**Demo 1：客服自动回复**

```text
客户消息 -> n8n Webhook -> Dify 判断意图 -> 生成回复 -> 推送到飞书群 / 邮件
```

**Demo 2：表格数据处理**

```text
上传 Excel / CSV -> n8n 解析 -> Dify 分类 / 总结 -> 输出处理结果
```

**Demo 3：自动日报生成**

```text
定时触发 -> 抓取指定数据源 -> Dify 汇总 -> 生成日报 -> 发送到飞书 / 邮件
```

#### 销售素材

- [ ] 3 个 Demo 演示视频（手机录屏即可，配文字说明）
- [ ] 3-5 个服务帖标题
- [ ] 1 份报价表
- [ ] 1 页服务介绍图

## Month 1：拿到第一批客户和案例（正式推广第 1-4 周）

### Month 1 目标

- 第一目标：3 个真实案例
- 第二目标：¥2000-5000 收入
- 超预期目标：¥8000+
- 核心判断标准：是否有客户愿意为“省重复工作”付钱

### Week 1：发帖 + 内容 + 承接咨询

#### 每日动作

```text
上午：闲鱼 / 猪八戒发或刷新服务帖，优化标题和详情页
中午：回复咨询，记录需求
下午：小红书发 1 条 Demo / 效果展示
晚上：微信群发言，贡献案例和经验，承接私聊
```

#### 渠道策略

| 渠道 | 做法 |
|------|------|
| 闲鱼 | 发布 / 刷新服务帖，测试标题关键词，回复主动咨询 |
| 猪八戒 | 发布服务，投标明确需求的项目 |
| 小红书 | 发 Demo 视频、前后效果对比、工具使用截图 |
| 知乎 | 回答“AI 提效”“n8n 能做什么”“企业知识库怎么搭”类问题 |
| 微信群 | 先分享案例和经验，不硬广；有人互动后承接私聊 |
| 现有工具用户 | 对付费用户和批量上传用户做 B2B 转化 |

#### 私聊来源优先级

1. 电商文案 / 简历工具的付费用户
2. 闲鱼 / 猪八戒主动咨询的人
3. 小红书评论、收藏、私信后有互动的人
4. 微信群里发言后加微信的人
5. 完全陌生冷私信（低优先级）

#### B 计划

如果 Week 1 没有有效咨询：

- 主动找 3 个有明确需求的人免费做一个小流程
- 只做 1 个流程，不接大系统
- 必须换取匿名案例、截图和评价

## 决策日志

| 日期 | 决策 | 原因 |
|------|------|------|
| 2026-05-20 | 主攻 n8n + Dify 工作流外包 | 最容易成交，客户能理解“省重复工作” |

## 风险与应对

| 风险 | 概率 | 应对 |
|------|------|------|
| Week 1 无人咨询 | 中 | 主动找 3 人免费做，强制制造第一个案例 |
